Freihandlinie zeichnen

Zeichnen von Freihandlinien.

So wird's gemacht:

§Wählen Sie im Funktionsbereich eine Stiftnummer:

§Klicken Sie am Startpunkt auf die linke Maustaste.

Wenn Sie die Maus bewegen, wird die Spur aufgezeichnet. Wenn Sie sich mit gedrückter linker Maustaste über die Zeichnung bewegen, wird der Schnellzoom ausgelöst.

§Markieren Sie den Endpunkt, indem Sie erneut auf die linke Maustaste klicken.

Die Linie wird gezeichnet.
