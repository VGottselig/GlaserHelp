Dübelleiste beschriften

Alle Inhalte aufklappen/schließen

1.Dübelleiste auswählen und Beschriftung ausrichten

§Klicken Sie eine Dübelleiste an. [Welches Element (anklicken)].

§Geben Sie einen Winkel ein oder übernehmen Sie einen vorgeschlagenen Winkel.

Optionen

Schnellauswahl. Drehung des Elements um die Systemlage (Systemwinkel beachten).

 

Ausrichtung der Element-Grundlinie:

P...: Parallel zu einem Bezugselement.

O...: Orthogonal zu einem Bezugselement.

§Klicken Sie entweder P... oder O... und anschließend ein beliebiges Zeichenelement an.

Siehe auch: Ausrichtungsoptionen

2.Beschriftung platzieren

§Entscheiden Sie, ob Sie nach dem Absetzen der Beschriftung Zeigerlinien hinzufügen möchten. [Lage].

Im Funktionsbereich:

Ohne Zeigerlinien.

Wählen Sie diese Funktion, wenn alle Dübelleisten vom gleichen Typ sind. In diesem Fall sind Zeigerlinien nicht erforderlich.

Mit Zeigerlinien.

Wählen Sie diese Funktion, wenn z. B. Zweier- und Dreierelemente zusammengestellt werden. Mit Hilfe der Zeigerlinien wird deutlich, wo welche Typen eingebaut werden sollen.

§Führen Sie einen der folgenden Schritte aus:

a)Bewegen Sie den Text an die gewünschte Stelle und legen Sie ihn mit Linksklick ab.

b)Bearbeiten Sie die Beschriftung, indem Sie unter der Abfrage auf klicken. Sie können die Beschriftung mit individuellen Zusatztexten oder mit der Angabe des Bauteilfaktors ergänzen.

3.Zeigerlinien hinzufügen (optional)

§Unter der Abfrage Zeigerlinie können Sie wählen, ob die Zeigerlinie vom Textanfang oder vom Textende bis zur Cursorposition gezeichnet werden soll.

Sie können jederzeit zwischen den Optionen wechseln.

§Klicken Sie die Elemente an, an die Zeigerlinien gezogen werden sollen.

Zeigerlinien können nur zu passenden Elementen erzeugt werden.

§Beenden Sie das Zeichnen der Zeigerlinien mit Rechtsklick.

Zugehörige Aufgaben:

·Beschriftung löschen
