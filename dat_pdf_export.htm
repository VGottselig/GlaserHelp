PDF exportieren

Erstellen einer PDF-Datei aus der aktuellen Zeichnung. Die Zeichnung wird als vektorisierte PDF-Datei exportiert, sodass Sie die Datei wieder in ISBCAD einlesen und enthaltene Linien, Kreise, Texte usw. weiterbearbeiten können.

1.Zeichenelemente auswählen

Führen Sie einen der folgenden Schritte aus:

a)Wählen Sie im Funktionsbereich die Funktion Fest. [Rechteck verschieben oder Faktor eingeben].

Schieben Sie das Rechteck um die Zeichenelemente. Wenn Sie einen Faktor eingeben, wird das Rechteck entsprechend vergrößert oder verkleinert.

b)Wählen Sie im Funktionsbereich die Funktion Belieb. [1.Punkt]; [2.Punkt].

Ziehen Sie einen Rahmen um die Zeichenelemente.

c)Wählen Sie im Funktionsbereich die Funktion Plotfl.

Die gesamte Plotfläche wird ausgegeben.

d)Wählen Sie im Funktionsbereich die Funktion RaAusg und wählen Sie die Blattrahmen, die ausgegeben werden sollen.

2.Datei für die Konvertierung anpassen und speichern

Nach Auswahl der Zeichenelemente wird der Dialog ISBCAD Druckoptionen geöffnet.

§Wählen Sie im Dialog die Einstellungen für die Ausrichtung und die Anzahl der Exemplare.

Die Vorschau zeigt das Druckergebnis für die aktuellen Einstellungen.

§Klicken Sie auf Weiter.

Der Speichern-Dialog wird geöffnet. Speichern Sie die PDF-Datei unter Angabe des Dateinamens und des Speicherortes.

Tipp:

 

Praktisch ist die Ausgabe mit jedem Programm möglich, welches PDF-Dateien erzeugt. Die Benutzung des eDocPrinter PDF Pro GLASER hat aber entscheidende Vorteile.

·Das Papierformat passt sich immer der Plotgröße an.

·Die Folien können im PDF-Format übernommen werden.

Planausgaben können ab Acrobat-Reader© 7.0 oder höher mit dem Elementstatus "enthält Ebenen" eingelesen werden. Diese Ebenen entsprechen zu 100% den ISBCAD Folien. Der Name setzt sich aus der Foliennummer und dem Namen zusammen. Ausgeblendete Folien werden nicht ausgegeben.
