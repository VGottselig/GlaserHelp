Optionen im Dialog "Plotter-Konfiguration"

Drucker

Wenn Sie auf Ihrem Betriebssystem einen Drucker eingerichtet haben, können Sie hier den Druckertreiber auswählen.

Name

Auswahl eines auf Ihrem System verfügbaren Drucker- oder Plottertreibers. Klicken Sie auf , um die Dropdown-Liste mit allen eingerichteten Treibern zu öffnen. Klicken Sie anschließend den entsprechenden Treiber an.

Ruft den Einstellungsdialog des Gerätetreibers auf.

Ausgabeprofil

Stellen Sie hier für jeden Bedarf speziell definierte Ausgabeprofile ein. Sie können z. B. ein Profil erstellen, in dem alle Linien, Kreise und Texte in Schwarz ausgegeben werden.

Name

Auswahl eines Ausgabeprofils. Sie können ein anderes Profil wählen, indem Sie die die Dropdown-Liste [] aufschlagen und das Profil anklicken. Mit Rechtsklick auf das Listenfeld öffnen Sie ein Kontextmenü mit Bearbeitungsoptionen.

Kopie eines Profils erstellen

1.Wählen Sie aus der Dropdown-Liste das zu kopierende Profil aus.

2.Öffnen Sie mit Rechtsklick das Kontextmenü und wählen Sie Profil kopieren.

3.Öffnen Sie erneut das Kontextmenü und wählen Sie Profil einfügen.

4.Benennen Sie das Ausgabeprofil um und bestätigen Sie mit der Eingabetaste.

Die Kopie des Profils wird mit dem angegebenen Profilnamen angelegt.

Profil löschen

Löscht das Profil ohne Rückfrage.

Assistent

Öffnet den Profilassistenten zum Einrichten der Profileigenschaften.

Profil umbenennen

Die Bezeichnung des aktuell markierten Profils kann direkt geändert werden. Wählen Sie das entsprechende Profil aus, benennen Sie es um und bestätigen Sie mit der Eingabetaste.

Es erscheint eine Meldung, in der Sie entscheiden können, ob Sie den Assistenten zum Einrichten eines neuen Ausgabeprofils benutzen möchten oder nicht. Wenn Sie Ja klicken, wird der Dialog Profilassistent geöffnet (» Profilassistent). Wählen Sie im Dialog die entsprechenden Optionen. Nach der Fertigstellung wird das Ausgabeprofil unter Name und im Funktionsbereich zur Auswahl angeboten.

Treffen Sie in den Auswahlboxen für Breite und Höhe durch Anklicken der vorgeschlagenen Maße (DIN-Formate) Ihre Vorauswahl. Die Ausgabegröße sollte den verfügbaren Druckbereich nicht überschreiten.

Steuerung

WINDOWS GDI:

Wenn die Zeichnung WINDOWS® True‑Type Fonts, Dokumente und/oder Pixelbilder enthält, muss diese Einstellung gewählt werden.

ISBCAD Direktmodus:

Nur bei Plottern möglich, wenn keine WINDOWS®-Schriften bzw. Bitmaps auf der Zeichnung vorhanden sind. Ist das trotzdem der Fall, werden die WINDOWS®-Fonts in ISBCAD Vektorfonts umgewandelt und statt der Bilddatei wird nur ein ausgekreuzter Rahmen mit dem Bilddateinamen geplottet.

In der Ausklappliste kann ein auf Ihrem System verfügbarer Gerätetreiber gewählt werden. Bei der Einrichtung des Plotters in den WINDOWS®-Systemeinstellungen sollte Universal  nur Text ausgewählt werden.

Skalierung

Automatisch berechnen/anpassen

Einstellung für Drucker.

Die Größe der Zeichnung bzw. des Zeichnungsausschnitts [Datei | Drucken | Plotfläche, fester Ausschnitt bzw. beliebiger Ausschnitt] wird immer auf die Größe für festen Ausschnitt skaliert.

Nur wenn der Faktor 1.0 beträgt, wird der Ausschnitt bzw. die Zeichnung in dem Maßstab ausgegeben, in dem sie im Konstruktionsprogramm gezeichnet wurde.

Ist der Faktor <1.0 wird die Zeichnung verkleinert.

Manuell eingeben / maßstäblich

Einstellung für Plotter.

Tragen Sie in das darunterliegende Eingabefeld den Faktor für die Zeichnungsgröße ein. Im Normalfall ist der Faktor 1!

Auflösung für gefüllte Flächen:

Hoch, Normal, Niedrig

Mit diesen Optionen bestimmen Sie, wie genau die Füllung an den Flächengrenzen gezeichnet werden soll. Wenn Sie "niedrig" wählen, kann es vorkommen, dass die Füllung nicht exakt in den Flächengrenzen liegt, sondern diese etwas überschreitet.

Einstellungen

Strichbreite skalieren bei Faktor <> 1

Wird der Plan bei der Ausgabe vergrößert oder verkleinert, werden die Strichbreiten mit dem gleichen Faktor verändert.

180° drehen

Die Zeichnung wird um 180° gedreht ausgegeben.

Stiftsortierung rückwärts

Bei der Ausgabe werden die Stifte von hinten nach vorne aufgenommen. Damit werden die Linien der Geometrie und Schraffur, die üblicherweise mit niedrigen Stiftnummern gezeichnet werden, über den Füllungen sichtbar.

ohne Sortierung

Die Ausgabe erfolgt in der Reihenfolge, in der die Zeichnung erstellt wurde.

ohne Diagonalpunkte

Standardmäßig werden unten links und oben rechts kleine Markierungen gesetzt. Damit stellen Sie sicher, dass der Plan in voller Größe geplottet wird, wenn kein Rand eingestellt ist. Einige Plotter verschieben sonst die Zeichnung an den Rand.

Papierquelle automatisch

Es wird das günstigste Papierformat aus der zur Verfügung stehenden Liste gewählt.

90° Drehung vorschlagen

Soll immer eine Drehung um 90° vorgeschlagen werden, setzen Sie einen Haken in die Checkbox.

0° Drehung vorschlagen

Die Drehung um 0° wird immer vorgeschlagen.

Elemente nicht überprüfen

Vor der Ausgabe werden alle Elemente überprüft, ob sie ausgegeben werden oder nicht. Wenn z. B. Linien mit Stift 8 (keine Darstellung (Stifte/Farben)) gezeichnet wurden, werden Sie vor der Ausgabe darauf hingewiesen.

Exemplare vom Treiber ignorieren

Gibt es im Gerätetreiber [Drucker | Einstellungen] eine Vorgabe für die Exemplare, wird diese hiermit außer Kraft gesetzt.

Ausgabe Bitmaps

Folienstruktur für PDF erstellen

Diese Auswahlmöglichkeit erscheint nur, wenn Sie als Drucker eDocPrinter PDF Pro GLASER gewählt haben. Damit kann jeder, der die PDF-Datei öffnet, Folien aus- und einschalten. Das kann hilfreich sein, wenn die Folien mit Schraffuren ausgeblendet werden können. Für das Format PDF/A-1 ist diese Option nicht verfügbar!

Angaben zur Darstellung der Linienenden. Öffnet den Dialog Linienattribute (Caps).

Überstand am Linienende

Voreinstellung: rund. Bei sehr starker Vergrößerung können Sie die Unterschiede im Ausdruck erkennen. Die gewählte Art wird in der Skizze angezeigt.

Linienattribute übernehmen:

Soll der Ausdruck mit diesen Attributen erfolgen, setzen Sie einen Haken in die Checkbox.
Optionen

Einstiftplotter

Auf der Seite für die Treiber müssen Sie den ISBCAD Direktmodus und als Gerätetreiber einen Stiftplotter wählen.

als Einstiftplotter verwenden:

Statt mit mehreren verschiedenen Stiften zu arbeiten, können auch mit nur einem Stift verschieden breite Linien gezeichnet werden.

Abrückmaß:

Damit unterschiedlich breite Linien gezeichnet werden, muss der Stift mehrfach die Linie mit seitlichem Versatz zeichnen.
Geben Sie das Maß in 1/100 mm an.

Schnittstelle für Ausgabe

Anschluss von Drucker:

Standardeinstellung.

Lokale Schnittstelle:

In äußerst seltenen Fällen ist der Plotter an eine lokale, serielle Schnittstelle (z. B. Com1) angeschlossen. Diese kann über das Auswahlfeld gewählt werden.

Wenn Sie über die lokale Schnittstelle (Com1 / Com2) ausgeben, muss diese konfiguriert werden. Lesen Sie dazu im Handbuch Ihres Plotters nach.
