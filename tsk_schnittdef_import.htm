Schnittdefinition exportieren

IFC Section Tool Menüleiste → Datei → Schnittdefinition: Export

Sie können die unter Schnitte angezeigten Schnitte in einer Datei (*.ifcschnitte) exportieren. Das ist praktisch, wenn Änderungen in einer IFC-Datei zu erwarten sind. So lassen sich die bereits erzeugten Schnitte wieder einlesen, an die geänderte Geometrie anpassen und in eine bestehende AKT-Zeichnung übernehmen. Sie können eine gespeicherte Schnittdefinition weitergeben, um sie an einem anderen Rechner einzulesen und sie in derselben IFC-Datei weiterbearbeiten.

Die Schnittdefinition wird mit der Identifikationsnummer (GUID) der IFC-Datei gespeichert. Die GUID stellt sicher, dass nach Änderungen in der IFC-Datei, dem Kopieren oder einer Dateiumbenennung, diese wieder von einem Programm identifiziert werden kann. Daher lässt sich eine Schnittdefinition im IFC Section Tool nur einlesen, wenn sie zur korrespondierenden IFC-Datei passt.

Diese Informationen werden gespeichert:

·GUID der IFC-Datei

·Schnittausrichtung [X, Y, Z, Frei]

·Schnittposition im Modellbereich

·Bezeichnung, Maßstab, Schnitttiefe

·Status: Eingeblendet oder ausgeblendet

·Bei Horizontalschnitten [Z]: Status der Option Z-Schnitt im Grundriss spiegeln

So wird's gemacht:

§Klicken Sie in der Menüleiste auf [Datei → Schnittdefinition: Export] oder drücken Sie Strg+O.

§Wählen Sie das Verzeichnis, in dem die Schnittdefinition gespeichert werden soll.

Vorgeschlagen wird das Verzeichnis, in dem die IFC-Datei liegt.

§Geben Sie den Dateinamen an.

Vorgeschlagen wird der Dateiname der IFC-Datei.

§Klicken Sie abschließend auf Speichern.

Die Schnittdefinition wird im gewählten Verzeichnis abgelegt.

Zugehörige Aufgaben:

·Schnittdefinition importieren

·Schnitt erzeugen und bearbeiten
