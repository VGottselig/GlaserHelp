Drucken eines Artikels

In der Kontexthilfe können Sie jederzeit ein oder mehrere Artikel auf ihrem Drucker oder in einer PDF-Datei ausgeben.

Voraussetzung für das Drucken oder Ausgeben in einer PDF-Datei ist, dass Sie die entsprechenden Druckertreiber auf Ihrem PC installiert und eingerichtet haben.

So drucken Sie einen Artikel

§Markieren Sie den Artikel , der gedruckt werden soll, indem Sie diesen auf der Registerkarte Inhalt anklicken.

Der markierte Artikel wird blau hinterlegt.

Bei mehreren Artikeln: Klicken Sie auf der Registerkarte Inhalt einen beliebigen Artikel innerhalb des Kapitels an, das gedruckt werden soll.

Alternativ können Sie das entsprechende Kapitel auch auswählen, indem Sie auf das Buch-Icon im Inhaltsverzeichnis klicken.

§Rufen Sie per rechten Mausklick auf den Artikel das Kontextmenü auf.

Das Kontextmenü wird mit dem Eintrag Drucken geöffnet.

§Wählen Sie im Dialog Thema drucken die Option Ausgewähltes Thema drucken aus. Bestätigen Sie mit OK.

Der Dialog Drucken wird geöffnet. Hier können Sie den gewünschten Drucker auswählen und ggf. die Druckereinstellungen konfigurieren. Alternativ können Sie den Dialog Drucken auch in der Menüleiste über das Drucken-Symbol öffnen.

Bei mehreren Artikeln: Wählen Sie im Dialog Thema drucken die Option Ausgewähltes Thema und alle Unterthemen drucken aus. Unabhängig davon, welchen Artikel Sie ausgewählt haben, wird das komplette Kapitel ausgedruckt.

§Klicken Sie auf Drucken, um die Druckdaten an den Drucker zu senden.

Falls Sie eine PDF-Datei erzeugen, geben Sie den Speicherort der PDF ein und bestätigen Sie diesen.
