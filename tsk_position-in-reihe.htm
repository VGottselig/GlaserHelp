Positionen in Reihe bringen

Wenn Sie die Nummerierung nach der Ausrichtung vornehmen wollen, werden alle Positionen neu gezählt, beginnend mit der angegebenen Himmelsrichtung und anschließend im Uhrzeigersinn.

NW = Nordwesten

N = Norden

NO = Nordosten

W = Westen

X = keine Himmelsrichtung

O = Osten

SW = Südwesten

S = Süden

SO = Südosten

·Die Nummerierung erfolgt bauteilbezogen: D1, D2, D3 oder F1, F2, F3 ...

·Die vorhandene Reihenfolge wird beibehalten, wobei die niedrigste Nummer zur Nummer "Eins" wird. Alle Leerpositionen werden entfernt. Die Flächen ohne Ausrichtung bleiben immer am Schluss stehen.

§Klicken Sie unterhalb der Abfrage die Himmelsrichtung an, von der aus nummeriert werden soll. [Mit welcher Himmelsrichtung beginnen].

Die Positionsnummern werden aktualisiert.
