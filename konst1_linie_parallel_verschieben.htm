Linie parallel verschieben

Mit dem Verschieben der Linie werden entweder nur die an den Endpunkten anschließenden Linien oder alle auf der Linie liegenden Punkte mitgeführt.

So wird's gemacht:

§Klicken Sie die zu verschiebende Linie auf der Seite an, zu der die Verschiebung ausgeführt werden soll. [Welche Linie(n)].

Mit Retan können Sie mehrere Linien auswählen. Die Richtung wird aus der zuerst angeklickten Linie ermittelt. Wenn Sie die zu verschiebenden Linien ausgewählt haben, bestätigen Sie mit Rechtsklick.

§Wählen Sie, wie die Linienpunkte verschoben werden sollen:

Es werden alle auf der Linie liegenden Punkte verschoben.

Es werden nur die Punkte, die auf den Endpunkten liegen, verschoben.

[Wert eing.]

Eingabe eines Wertes.

§Geben Sie den Verschiebebetrag ein und bestätigen Sie die Eingabe mit der Eingabetaste. Für Verschiebungen in Richtung der negativen Koordinatenachsen geben Sie den Betrag mit negativem Vorzeichen "-" an.

[Punkt andr.]

Übernahme eines Wertes oder Eingabe einer Differenz.

Ohne Differenzangabe (Retaus)

§Klicken Sie einen Punkt an, dessen Koordinate Sie übernehmen möchten.

Mit Differenzangabe (Retan)

§Klicken Sie die Bezugslinie an und geben Sie die Differenz mit dem entsprechenden Vorzeichen ein. Bestätigen Sie die Eingabe mit der Eingabetaste.

[frei]

Freies Verschieben. Wenn Sie den Cursor bewegen, wird Ihnen eine Vorschau angezeigt.

§Bewegen Sie den Cursor an die gewünschte Stelle und klicken Sie einen beliebigen Punkt auf der Zeichenfläche oder einen Fangpunkt an.

Die zuerst angeklickte Linie wird parallel verschoben. Alle anderen markierten Punkte werden mit dem gleichen Wert verschoben.

Beispiel: Linien parallel verschieben

Alle Punkte: Alle Punkte, die auf der (den) angeklickten Linie(n) liegen, werden mitgenommen.

Nur Endpunkte: Die angeklickte Linie wird parallel verschoben. Die an den Enden anliegenden Linien werden mitgezogen.

Beispiel: Das Verschieben der Deckenoberkante bewirkt die automatische Anpassung der Wandfußpunkte.
