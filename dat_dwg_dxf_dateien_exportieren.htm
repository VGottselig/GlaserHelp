DWG/DXF Dateien exportieren

Alle Inhalte aufklappen/schließen

Speichern der aktuellen Datei im DXF- oder DWG-Format. Öffnet den Dialog AutoCAD-Export.

Wenn Sie mehrere DWG/DXF-Dateien gleichzeitig umwandeln möchten, verwenden Sie das Modul [DXF/DWG Export] aus dem ISBCAD Hauptmenü.

Datei für die Konvertierung anpassen und speichern

§Wählen Sie im Dialog AutoCAD-Export die gewünschten Einstellungen.

Siehe Dialogbeschreibung weiter unten.

§Bestätigen Sie die Einstellungen mit OK.

Der Dialog Speichern unter wird geöffnet.

§Geben Sie im Dialog den Dateinamen ein und wählen Sie unter Dateityp, in welchem Format die Datei gespeichert werden soll.

Faktor anhand des Maßstabs ermitteln

DWG/DXF-Dateien können beim Export nur in einen Maßstab umgewandelt werden. Über den Faktor geben Sie an, in welchem Maßstab die Elemente der Zeichnung exportiert werden sollen und damit auch ihre Länge. Beachten Sie daher, dass Elemente mit unterschiedlichen Maßstäben bei der Umrechnung mit demselben Faktor mit verschiedenen Längen gezeichnet werden.

Die Längen von Strecken in der exportierten DWG-Datei berechnen sich wie folgt:

LDWG = F * 100 * LAKT

Länge in DWG-Datei = Faktor * 100.0 * Länge in AKT-Datei, wobei die zugrunde liegende Maßeinheit Meter ist und in ISBCAD die Länge auf dem Papier zählt.

Beispiel:

Sie zeichnen in ISBCAD eine Linie im Maßstab 1:50 und geben als Länge den Wert 10 ein. Dann entspricht dies einer Strecke von 10.0 Metern auf der Baustelle und 0.2 Meter auf dem Papier.
Damit in der DWG-Datei alle im Maßstab 1:50 gezeichneten Elemente mit der richtigen Länge gezeichnet werden, müssen die Elemente mit dem "richtigen" Faktor exportiert werden.

Der Faktor für die Umrechnung berechnet sich wie folgt:

F = M / 100

Faktor = Maßstabszahl / 100. In diesem Beispiel wurde eine Linie im Maßstab 1:50 gezeichnet. Daraus ergibt sich: F = 50 / 100.

Die Zeichnung muss daher mit dem Faktor 0.5 exportiert werden, damit die Streckenlängen den realen Weltkoordinaten, wie sie in AutoCAD verwendet werden, entsprechen. Linien mit anderen Maßstäben werden mit demselben Faktor umgerechnet und erhalten dadurch eine falsche Länge.

Bemaßungstexte übertragen

Maßketten behalten in der AutoCAD-Datei die in ISBCAD berechnete Maßkettenzahl, unabhängig vom Maßstab.

Beachten Sie, dass die Maßketten erst korrigiert werden müssen, bevor Sie die Zeichnung weiterbearbeiten können. Mit der Übernahme der Maßkettenzahlen aus ISBCAD stimmen in AutoCAD die tatsächlichen Längen ggf. nicht mit den Maßkettenzahlen überein!

Maßkettenzahlen werden angepasst, wenn in der AutoCAD-Datei eine Maßkette verlängert oder verkürzt wird. In diesem Fall berechnet AutoCAD die Maßkettenzahl selbst.

Koordinaten transformieren

Normalerweise werden die Koordinaten des Blattursprungs (0, 0) als Bezugskoordinaten übergeben. Um die exportierte Zeichnung in ein vorhandenes Koordinatensystem einzupassen, stehen mehrere Möglichkeiten zur Verfügung.

Keine Transformation

Wählen Sie diese Option, wenn sich kein Koordinatensystem [Konst2 | Koorvm] auf der Zeichnung befindet oder keines der vorhandenen Koordinatensysteme benutzt werden soll.

Bezugssystem ist das erste gefundene

Alle Koordinaten werden auf das zuerst angelegte Koordinatensystem bezogen.

Bezugssystem auswählen

Während der Umformung erscheint ein Dialog, in dem Sie das Bezugskoordinatensystem auswählen können.

Sonstiges

Elemente außerhalb ignorieren

Außerhalb der Plotfläche liegende Elemente werden nicht mit übertragen.

Lage der Maßkettenpunkte beibehalten

Bleiben die Maßkettenpunkte an ihrem ursprünglichen Ort, werden die Maßhilfslinien in DXF/DWG bis in die Nähe dieser Punkte verlängert. Das heißt, dass die Maßketten ein anderes Aussehen annehmen. Die Maßkettenpunkte behalten dann aber ihre Funktionalität.

Verfügbare Export-Profile

Auflistung aller auf Ihrem System verfügbaren Export-Profile. Es können beliebig viele Profile gespeichert werden.

Die Export-Profile werden im aktiven ISBCAD Profil gespeichert.

Um ein Export-Profil zu aktivieren, klappen Sie die Dropdown-Liste auf und klicken Sie das entsprechende Profil an. Alternativ können Sie auch ein Profil in der Tabelle unter Profilname: anklicken und anschließend mit Profil aktivieren übernehmen.

Profilname

Bezeichnung des Profils. Kann über Profil ändern geändert werden.

Kommentar

Beliebiger Text zur Beschreibung des Export-Profils. Kann über Profil ändern hinzugefügt und geändert werden.

Übernimmt die Einstellungen des in der Liste markierten Export-Profils.

Öffnet den Dialog Profil ändern. Sie können den Profilnamen und Kommentar ändern.

Erstellt eine Kopie des markierten Profils. Mit der Schaltfläche Profil ändern können Sie einen neuen Profilnamen vergeben und ergänzend einen Kommentar hinzufügen.

Das markierte Profil wird gelöscht.

Es wird der Dialog Speichern unter geöffnet. Wählen Sie den Pfad und geben einen Namen an. Das gewählte Profil wird als XML-Datei gespeichert.

Im Dialog Öffnen können Sie den Pfad angeben, von wo ein externes Export-Profil eingelesen werden soll.
