IFC-Datei öffnen

IFC Section Tool Menüleiste → Datei → Öffnen

Das IFC Section Tool unterstützt das Einlesen von IFC-Dateien der Schemata IFC2x3 und IFC4. Nach dem Laden wird das 3D-Modell im Modellbereich angezeigt und kann direkt weiterbearbeitet werden.

·Es können IFC-Dateien mit den Dateierweiterungen (*.ifc) und (*.ifcZIP) eingelesen werden.

·Folgende IFC-Exporttypen werden unterstützt:

IFC-Format

Exporttypen

IFC2x3

·Coordination View 2.0

·Surface Geometry

·Steel Fabrication Type

·Coordination View 1.0

IFC4

·Reference View

·Design Transfer View

·Ifc4precast View

·Einschränkung beim Reference View: Aus dem 3D-Modell erzeugte Schnitte können unter Umständen unvollständig sein. Für ein besseres Ergebnis stellen Sie sicher, dass die IFC-Datei im Modellierungsprogramm mit dem Exporttyp Design Transfer View erzeugt wurde.

·Die Genauigkeit der Schnitte hängt direkt von der Qualität des zugrunde liegenden IFC-Modells ab. Je präziser die Bauteile im Modell definiert sind, desto genauer ist sowohl die Darstellung des 3D-Modells als auch der daraus abgeleiteten Schnitte. Fehlende oder unvollständige Bauteildefinitionen können dazu führen, dass Elemente im Schnitt nur teilweise oder gar nicht dargestellt werden.

·Wenn in einem einzulesenden IFC-Modell bereits Schnitte definiert wurden, werden diese in einem GLASER-Programmordner gespeichert:

(C:\Users\"Benutzer"\AppData\Roaming\GLASER Programmsysteme GmbH\ISBCAD 2026\Data\IFC-Schnitt-Tool)

Öffnen Sie dasselbe IFC-Modell in einer neuen ISBCAD-Instanz über das IFC Section Tool, werden zuvor erzeugten Schnitte mitsamt ihrer ursprünglichen Position im 3D-Modell wieder angezeigt. Da jede IFC-Datei eine eindeutige Identifikationsnummer hat, die GUID (eng. Global Unique Identifier), bleiben die Schnittdefinitionen auch nach dem Kopieren, Umbenennen oder Ändern der IFC-Datei erhalten. Siehe auch: Schnittdefinition exportieren

So wird's gemacht:

§Klicken Sie in der Menüleiste des IFC Section Tools auf [Datei → Öffnen] oder drücken Sie die Tastenkombination Strg+O.

§Durchsuchen Sie im Dialog das Dateisystem nach der zu öffnenden IFC-Datei (*.ifc) und markieren Sie diese.

§Öffnen Sie die Datei per Doppelklick oder klicken Sie auf Öffnen.

Das 3D-Modell wird eingelesen und im Modellbereich angezeigt. Je nach Größe des 3D-Modells kann der Einlesevorgang einige Zeit in Anspruch nehmen.
