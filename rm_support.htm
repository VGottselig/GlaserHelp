Support

Im Menüpunkt [Support] können Sie den ISBCAD Kundensupport per E-Mail kontaktieren, Verbesserungsvorschläge auf der Plattform ISBCAD.NOLT eintragen, und den TeamViewer Support nutzen.

Eintrag

Beschreibung

E-Mail an Support senden

Kontaktaufnahme mit dem ISBCAD Kundensupport per E-Mail. Die aktuelle Zeichnung, das Profil und eine Protokolldatei können optional als Anhang hinzugefügt werden.

Siehe: E-Mail an Support senden

ISBCAD.NOLT - Ideenportal für ISBCAD

Öffnet die ISBCAD.NOLT-Feedback Plattform im Standardbrowser zum Eintragen von Verbesserungsvorschlägen für ISBCAD.

TeamViewer Support

Mit dem TeamViewer besteht die Möglichkeit, zwei Rechner über das Internet zu verbinden. Damit können wir Sie bei Bedarf per Fernzugriff bei der Problemlösung unterstützen. Starten Sie den TeamViewer erst dann, wenn Sie von unserem Support dazu aufgefordert werden.

 

ISBCAD Kundensupport

Wenn Sie bei der Verwendung von ISBCAD Fragen oder technische Probleme haben, stehen Ihnen verschiedene Support-Kanäle zur Verfügung.

Sie erreichen unseren Kundensupport von Montag bis Freitag von 09:00 - 12:00 Uhr sowie von 14:00 - 17:00 Uhr unter +49 511 592931-0, oder per E-Mail an: support@glasercad.de
