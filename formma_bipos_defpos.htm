Positionierungsstil konfigurieren

Alle Inhalte aufklappen/schließen

Einstellungen für die Darstellung der Formstahl- und Formmattenpositionierungen. Öffnet den Dialog Einstellungen für Positionierung mit der Registerkarte Formmatten.

Optionen im Dialog "Einstellungen für Positionierung"

Formstahl

Konfiguration der Formstahl-Positionierungsstile (IDs).

§Wählen Sie unter Positionierungstext ID-Nr. (im unteren Dialogbereich) die ID, deren Einstellungen Sie ändern wollen.

Der ID-Name der gewählten ID wird unter der Vorschau angezeigt.

§Wählen Sie die gewünschten Darstellungseigenschaften (s. Optionen).

Änderungen einzelner Attribute werden sofort in der Vorschau angezeigt.

§Geben Sie im Auswahlfeld für ID-Name einen möglichst selbsterklärenden Namen ein (optional).

Im Auslieferungszustand haben alle IDs den Namen "Standard".

§Klicken Sie auf OK, um die Einstellungen zu speichern.

Der Dialog wird geschlossen; die Darstellung der Formstahl-Positionierungen kann über die Stilauswahl im Funktionsbereich für Formstahl [Formst | BiPos] gewählt werden.

Optionen

Positionierungstext

Textgröße

Text innerhalb des Positionskreises bzw. Rechtecks
Wählen Sie aus der Liste einen der vorgeschlagenen Werte aus oder tippen Sie einen beliebigen Wert ein.

Stiftnummer

Sie können eine der 256 möglichen Stiftnummern auswählen. Die Einstellung dafür wird in [Datei | Drucken/Plotten | Einstellungen] vorgenommen.

Breitenfaktor

Geben Sie einen Wert für den Breitenfaktor des Positionstextes an.

Font

Wählen Sie eine der zur Verfügung stehenden Schriftarten aus.

Faktor Radius Positionskreis

Bestimmt die Größe des Positionskreises. Es wird der Abstand vom Positionstext zum Kreis angegeben.

Positionslinie

Stiftnummer

Auswahl einer Positionslinie.

Ergänzungstext

Textgröße

Je nachdem, mit welchen Parametern (Anzahl oder Abstand) Sie die Verlegung durchgeführt haben, werden als Ergänzungstext Anzahl und Durchmesser und ggf. der Abstand angeschrieben.

Strichstärke

Wählen Sie einen Stift für den Text aus.

Breitenfaktor

Geben Sie einen Faktor für die Breite der Buchstaben ein. Um die Textbreite so klein wie möglich zu halten, ist ein Wert kleiner 1 sinnvoll.

Font

Sie können eine Schriftart wählen, die sich von der Schrift der Positionsnummer unterscheidet, z. B. Schriftart für Positionstext: Arial / Schriftart für Ergänzungstext: ArialNarrow.

Tipps:

 

·Um die Textbreite so klein wie möglich zu halten, ist ein Wert kleiner 1 sinnvoll.

·Das im Ergänzungstext verwendete Durchmesserzeichen (ø) kann über die Systemeinstellungen geändert werden. [Optionen | Systemdaten | Formstahl | Auszug Rundstahl → Zeichen für Bewehrungsdurchmesser].

Sonstiges

Abkürzung für Bügelabstand

Wenn der Ergänzungstext Angaben über den Abstand enthält, können Sie dafür ein Kürzel (z. B. "s=...") wählen. Sollte kein Platz mehr vorhanden sein, können Sie darauf verzichten, indem Sie das Feld leer lassen. Dann wird nur der Abstandswert geschrieben.

Trennzeichen

Wählen Sie ein Zeichen für die Trennung zwischen dem Durchmesser und dem Abstand.

Ohne Leerzeichen

Zwischen der Anzahl und dem Durchmesserzeichen und vor dem Durchmesserwert werden normalerweise Leerzeichen eingefügt. Wenn diese entfallen sollen, setzen Sie einen Haken.

Hochzahl ab Nachkommastelle

Wählen Sie, ab welcher Nachkommastelle eine Hochzahl eingefügt werden soll.

ID-Name

Angabe des Namens für die unter der Option Positionierungstext ID-Nr. gewählten ID.

Symbole für Ende und Mitte der Positionierung (Harfe)

Strichstärke

Wählen Sie eine der möglichen Stiftnummern aus.

Länge der Pfeil­spitze (1/10mm)

Geben Sie das Maß für die Länge des Endpfeils an.

Höhe der Pfeil­spitze (1/10mm)

Geben Sie einen Wert für die Höhe einer Pfeillinie an.

Durchmesser Kreis (1/10mm)

Geben Sie ein Maß für die Größe des Kreises an.

Überstand (1/10mm)

Bestimmen Sie die Länge der Striche und Schrägstriche.

Positionierungstext ID-Nr.

Auswahl des zu konfigurierenden Positionierungsstils. Eine Bezeichnung für die ID-Nr. können Sie unter ID-Name angeben.
