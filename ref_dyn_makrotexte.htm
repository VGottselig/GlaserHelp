Dynamische Makrotexte (Referenz)

ISBCAD bietet verschiedene vorgegebene Makrotexte, die Sie in Texteingaben einfügen können, z. B. das letzte Speicherdatum der Zeichnung.

Verfügbare dynamische Makrotexte

Jeder Makrotext wird von ISBCAD automatisch aktualisiert, sobald eine Änderung stattfindet, z. B. wenn Sie die Datei unter einem neuen Dateinamen oder an einem neuen Speicherort speichern.

Formatierung bearbeiten

Die Makrotexte {Today}, {Save Date} und {Create Date} werden von ISBCAD standardmäßig im Datumsformat "dd.MM.yyyy" ausgegeben (Tag.Monat.Jahr). Der Makro {Paper Size} wird in Zentimetern ausgegeben.

In Eingaben werden die Makrotexte in der Form {Save Date$dd.MM.yyyy} mit speziellen geschweiften Klammern gekennzeichnet. Sie können das Format von Makrotexten bearbeiten, indem Sie die Zeichenfolge nach dem $-Zeichen ändern.

Datumsformate

Ein Beispiel: Das Datumsformat {Save Date$dd. MMMM yyyy} wird als 15. Juli 2024 angezeigt. Wenn Sie das Format in {Save Date$yyyy-MM-dd} ändern, wird das Datum als 2024-07-15 angezeigt.

Blattformate

Ein Beispiel: Die Blattgröße {Paper Size$mm} wird als 1189/841 mm angezeigt. {Paper Size$DIN} gibt das DIN-Format aus: DIN A0.
