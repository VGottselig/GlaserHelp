Sparrenbild oder Einzelsparren ändern

Einzelne Sparren oder ganze Sparrengruppen ändern.

Wenn Sie einzelne Sparren verschoben oder in der Breite geändert haben, gehen diese Änderungen verloren.

So wird's gemacht:

§Wählen Sie, ob die Sparren einzeln oder als Gruppe geändert werden sollen. [Sparren/Sparrenbild wählen].

Ändern eines einzelnen Sparrens.

§Klicken Sie eine Linie eines Einzelsparrens an. Der Sparren wird markiert.

§Geben Sie die neue Breite ein und bestätigen Sie mit der Eingabetaste. [neue Breite]. Die Breite wird angepasst.

Ändern einer Sparrengruppe.

§Klicken Sie eine Linie eines Sparrens an. Die Gruppe wird gelöscht. Das Hilfspolygon wird wieder dargestellt.

§Richten Sie die Sparren neu aus oder übernehmen Sie die Ausrichtung mit Rechtsklick. [Verlegerichtung]. Nach dem Ausrichten wird der Dialog Sparrenverteilung geöffnet. Die Dialogbeschreibung finden Sie im Abschnitt Sparren verteilen.

§Ändern Sie im Dialog die entsprechenden Parameter und bestätigen Sie mit OK. [Verlegedialog]. Die Sparren werden den Angaben entsprechend geändert.
