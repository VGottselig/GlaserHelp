Aktualisierung auf Server-Betriebssystemen (Remote Desktop Services)

Auf Server-Betriebssystemen (Remote Desktop Services) werden Aktualisierungen nicht automatisch heruntergeladen. Es wird jedoch automatisch und regelmäßig auf neue Aktualisierungen geprüft. Wenn auf unserem Server eine neue Aktualisierung verfügbar ist, erhält der Anwender eine Meldung. In dieser wird der Anwender aufgefordert, den System-Administrator zu informieren.

Für die Aktualisierung benötigen Sie Administratorrechte.

Bevor Sie ISBCAD aktualisieren

Informieren Sie die aktuellen Nutzer, dass eine Aktualisierung bevorsteht. Die Anwender sollten alle Zeichnungen speichern und alle ISBCAD Instanzen beenden, auch das ISBCAD Hauptmenü.

So aktualisieren Sie ISBCAD auf einem Server-Betriebssystem

§Laden Sie die aktuelle Revision von unserem Download-Server herunter.

Zu unserer Website: https://www.glasercad.de/downloads/isbcad/

§Starten Sie die Installation durch Doppelklick auf Setup[.exe].

Setup führt die notwendigen Prüfungen durch und installiert die Aktualisierung.

Hinweise:

 

·Ggf. muss in der Firewall der folgende Prozess freigegeben werden.
(Der Prozess ist durch das GLASER Zertifikat "Code Signing" signiert.)

isbMaintSvc.exe:

Prüft auf Aktualität (Vergleich der Revisionsnummern), lädt die Aktualisierung vom GLASER Server herunter und stellt sie zur Installation bereit.

·Verwendet wird das HTTPS-Protokoll.
