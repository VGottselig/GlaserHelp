Volumen ermitteln

Alle Inhalte aufklappen/schließen

Es können Volumina von prismatischen Körpern ermittelt werden. Dabei wird die Grundfläche mit der Höhe multipliziert.

Hinweise:

 

·Falls die Kontur des Polygons aus unterschiedlichen Maßstäben besteht, werden Sie nach dem Bestätigen des Polygons in einer Meldung darauf hingewiesen. Ihnen werden alle gefundenen Maßstäbe angezeigt, sodass Sie für die Massenberechnung den korrekten Maßstab auswählen können. Wählen Sie aus der Dropdown-Liste den gewünschten Maßstab und bestätigen Sie mit OK.

Achtung! Der im Dialog gewählte Maßstab wird nur für die Berechnung verwendet. Die ursprünglichen Maßstäbe der Elemente werden nicht geändert!

·Die Darstellung der Massen-Angaben kann vorab über [Konst3 | Massen | Einstellung] konfiguriert werden.

·Die Ausgabeeinheit können Sie in den Systemdaten ändern. [Option | SysDat | Ausgabeeinheiten].

1.Polygon(e) abstecken

§Wählen Sie unterhalb der Abfrage, wie Sie das Volumen ermitteln wollen.

Punkte anklicken: Bestimmung eines Volumens durch Anklicken von Zeichenpunkten.

Klicken Sie die gewünschten Zeichenpunkte nacheinander an. Wenn Sie den ersten Punkt wieder anklicken, wird die Fläche geschlossen. [Klicken Sie Punkte an].

Tipp: Mit einem Rechtsklick werden der erste und letzte Punkt automatisch miteinander verbunden.

Elemente anklicken: Bestimmung eines Volumens durch Anklicken von Elementen.

Klicken Sie die gewünschten Elemente nacheinander an und bestätigen Sie die Auswahl durch Rechtsklick. [Klicken Sie Elemente an].

In ein Polygon klicken: Auswahl einer in der Zeichnung vorhandenen, geschlossenen Form (aus Linien und/oder Kreisen).

Klicken Sie in eine geschlossene Form. [Punkt wählen].

Massentext anklicken und Werte übernehmen: Angaben aus einer frei platzierten Massen-Angabe, einem Schwerpunkt oder aus einem Raumstempel übernehmen. [Massentext auswählen].

Klicken Sie eine frei platzierte Massen-Angabe (Fläche, Länge, Volumen, Gewicht), einen Schwerpunkt oder einen Raumstempel an.

Folgen Sie der Abfrage, um weitere Massen-Angaben zu ermitteln und diese zu addieren oder zu subtrahieren.

Das Ergebnis wird Ihnen neben der Abfrage in der aktuellen Ausgabeeinheit angezeigt.

2.Höhe des Körpers angeben

§Geben Sie die Höhe ein und bestätigen Sie mit der Eingabetaste. [Dicke]. Alternativ können Sie die Höhe aus der Zeichnung abgreifen. Klicken Sie dazu unterhalb der Abfrage auf [Aus Zeichnung], wählen Sie eine der Optionen und folgen Sie der Abfrage:

Länge aus Linie: Bestimmung der Dicke über die Länge einer Linie.

Klicken Sie eine Linie an, deren Länge Sie übernehmen möchten. [Welche Linie].

Länge aus Abstand Punkt/Linie: Definition der Dicke über den Abstand von einem Punkt zu einer Linie.

Klicken Sie zunächst einen Punkt an [Welcher Punkt] und dann eine Linie [Welche Linie].

Länge aus Abstand Punkt/Punkt: Definition der Dicke über den Abstand zwischen zwei Punkten.

Klicken Sie den ersten Punkt an [1. Punkt] und dann den zweiten Punkt [2. Punkt].

Länge aus Abstand Linie/Linie: Definition der Dicke über den Abstand zwischen zwei Linien.

Klicken Sie die erste Linie an [1. Linie] und dann die zweite Linie [2. Linie].

Sie können die Eingabe jederzeit abbrechen, indem Sie auf Abbruch klicken oder die Esc-Taste drücken.

§Um das Volumen positiv anzusetzen, klicken Sie unterhalb der Abfrage auf [+Add] oder klicken Sie [-Sub] für ein negativ angesetztes Volumen. [Positiv + Negativ -]. Bestätigen Sie mit Rechtsklick.

Sie können sofort ein weiteres Volumen ermitteln.

§Um die Polygoneingabe zu verlassen, drücken Sie die Eingabetaste, Rechtsklick oder .

3.Beschriftung hinzufügen

§Geben Sie eine Bezeichnung für die Fläche bzw. den Raum an und bestätigen Sie mit der Eingabetaste. [Bezeichnung].

-Sie können unterhalb der Abfrage aus der Dropdown-Liste eine vordefinierte Bezeichnung wählen und diese bei Bedarf weiterbearbeiten.

-Sie können einen Text auf der Zeichenfläche anklicken, um diesen in die Eingabe zu übernehmen.

-Sie können einen individuellen Text eingeben.

§Geben Sie unter Textwinkel die Ausrichtung des Massen-Textes (in Grad) an und bestätigen Sie mit der Eingabetaste oder Rechtsklick.

Um den Massen-Text orthogonal oder parallel an oder auf einer Linie zu platzieren, wählen Sie O oder P und klicken Sie die Linie an.

Optionen

Schnellauswahl. Drehung des Elements um die Systemlage (Systemwinkel beachten).

 

Ausrichtung der Element-Grundlinie:

P...: Parallel zu einem Bezugselement.

O...: Orthogonal zu einem Bezugselement.

§Klicken Sie entweder P... oder O... und anschließend ein beliebiges Zeichenelement an.

§Massen-Text platzieren [Textlage]:
Klicken Sie die Stelle an, an der der Massen-Text platziert werden soll.

Der Massen-Text wird in der Zeichnung angelegt.
