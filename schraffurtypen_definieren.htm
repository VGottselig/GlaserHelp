Schraffurtyp erzeugen

Sie können eigene Schraffuren auf Basis eines benutzerdefinierten Linienmusters erstellen. Ein Linienmuster kann aus einer Linie und aus mehreren zusammenhängenden Linien (Linienzug) bestehen. Beim Erzeugen werden diese Linien als Polygone betrachtet. Die Polygone werden einzeln abgefragt. Dabei geben Sie für jedes Polygon einen Abstand an, nach dem dieses horizontal und vertikal wiederholt werden soll. Nach dem Erzeugen und der Vergabe einer Schraffurbezeichnung wird der Schraffurtyp der Schraffurauswahl im Funktionsbereich hinzugefügt. Die Schraffuren werden im aktiven Profil gespeichert.

Gezeichnet: Doppellinie

Ergebnis: Neuer Schraffurtyp

Hinweis:

 

Schraffurtypen müssen im Maßstab 1 : 100 erzeugt werden. Beispiele zum Erzeugen von verschiedenen Schraffurtypen finden Sie weiter unten.

1.Schraffurteil zeichnen

§Wechseln Sie in [Konst 1 | Linien].

§Zeichnen Sie den Teil der Schraffur, der sich wiederholen soll.

§Wechseln Sie anschließend in [Konst1 | Schraf | DefSch].

2.Bereich für das Schraffurmuster wählen

§Wählen Sie die Zeichenelemente für das Schraffurmuster, indem Sie einen Rahmen um diese aufziehen. [1.Punkt]; [2.Punkt].

-Sie können nur einen Rahmen aufziehen. Bestimmen Sie den Bereich ggf. mit Elemente im Polygon [Polygo]. Die Anzahl der Polygone ist begrenzt. Wenn Sie zu viele Polygone einschließen, erscheint eine Meldung.

-Das Polygon, welches zuerst bearbeitet werden soll, wird innerhalb des Auswahlbereiches markiert.

3.Abstände für die Musterwiederholung angeben

Die Abstände werden grundsätzlich in der Eingabeeinheit Meter (m) eingegeben.

§Geben Sie unter X-Abst. n. Polyg. für jedes Polygon den Abstand in Metern ein, nach dem dieses horizontal wiederholt werden soll, und bestätigen Sie mit der Eingabetaste. Das aktuelle Zeichenelement ist markiert.

§Geben Sie unter Y-Abst. n. Polyg. für jedes Polygon den Abstand in Metern ein, nach dem dieses vertikal wiederholt werden soll, und bestätigen Sie mit der Eingabetaste.

Hinweis:

 

Bei Eingabe Y-Abstand = 0 wird eine durchgehende Linie erzeugt.

Tipp:

 

Geben Sie für Symbolschraffuren, wie Dämmung, einen größeren Abstand ein als die Symbole hoch sind.

4.Schraffur benennen

§Geben Sie unter Name eine Bezeichnung für den Schraffurtyp ein (maximal sechs Zeichen) und bestätigen Sie mit der Eingabetaste.

-Nach Bestätigung des Namens wird der Schraffurtyp der Schraffurauswahl hinzugefügt. Sie können sofort einen weiteren Schraffurtyp erzeugen.

-Wenn Sie einen vorhandenen Schraffurtyp ersetzen wollen, müssen Sie dessen Namen eingeben und die Rückfrage zum Ersetzen bestätigen.

-Im Dialog Schraffur können Sie den neuen Schraffurtyp konfigurieren.

Schraffurtyp erzeugen (Beispiele)

§Stellen Sie zunächst den Maßstab der Zeichnung auf 1 : 100.

§Wechseln Sie in [Konst 1 | Linien] und zeichnen Sie z. B. eine Doppellinie.

§Wechseln Sie anschließend in [Konst 1 | Schraf | DefSch] und ziehen Sie einen Rahmen um die Doppellinie. [1.Punkt / 2.Punkt].

§Folgen Sie anschließend den Abfragen und bestätigen Sie jede Eingabe mit der Eingabetaste.
