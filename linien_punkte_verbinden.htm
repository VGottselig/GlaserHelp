Linienpunkte verbinden

Verbinden beliebiger Zeichenpunkte mit einer Linie oder einem Linienzug.

Tipp:

 

Zum Verbinden lückenhafter Verbindungen können Sie auch die Funktionen "Punkt verschieben" [Konst1 | Linien | Puvers] oder "Schnittpunkt bilden" [Konst1 | Linien | Schnip].

So wird's gemacht:

§Wählen Sie unterhalb der Abfrage, ob Sie zwei oder mehrere Punkte verbinden wollen:

Verbindung zwischen zwei Punkten.
Nach Auswahl des zweiten Punktes [2. Linienpunkt] können Sie sofort eine weitere, von einem anderen Punkt ausgehende, Verbindung anlegen. [1. Linienpunkt].

Verbindung über beliebig viele Punkte. Nach Auswahl des zweiten Punktes [2. Linienpunkt] können Sie den nächsten Punkt anklicken. Die Abfrage [2. Linienpunkt] wird solange wiederholt, bis Sie die Auswahl mit Rechtsklick bestätigen.

§Klicken Sie die Punkte, die verbunden werden sollen, nacheinander an.

Vom jeweils letzten Punkt wird eine Voransicht zur aktuellen Cursorposition gezeigt.
