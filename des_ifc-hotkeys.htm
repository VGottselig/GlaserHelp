Hotkeys für Standardansichten

Sie können eine Tastenkombination nutzen, um eine bestimmte orthogonale Ansicht oder eine isometrische Ansicht einzustellen. In der Tabelle finden Sie die Hotkeys, mit deren Hilfe Sie die gewünschte Ansicht schnell abrufen können:

Taste

Ansicht von:

Alt + O

oben

Alt + U

unten

Alt + V

vorne

Alt + H

hinten

Alt + L

links

Alt + R

rechts

Alt + I

isometrisch

Alternativ lässt sich die Ansicht auch über die Menüleiste unter [Ansicht → Standard-Ansichten] einstellen.
