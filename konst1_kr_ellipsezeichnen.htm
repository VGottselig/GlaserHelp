Ellipse zeichnen

Alle Inhalte aufklappen/schließen

Zeichnen von polygonal angenäherten Ellipsen.

Hinweis:

 

Ellipsen bestehen aus einzelnen Liniensegmenten. Je höher die Anzahl der Liniensegmente, desto höher ist die Genauigkeit bei der Darstellung der Ellipse. Die Anzahl der Linien, aus denen eine Ellipse besteht, können Sie in den Systemdaten einstellen. [Option | SysDat → Konstruktionsprogramm → Genauigkeit für Ellipsen].

Strichtyp und Liniendarstellung (Stift) können über die Optionspaletten (im Funktionsbereich) gewählt oder mit der so wie-Funktion von einem anderen Element übernommen werden.

Stiftnummer, Strichtyp ändern und so wie-Funktion

Strichtyp-Auswahl.

so wie-Funktion: Auswahl der Linien- und Stifteinstellungen entsprechend eines anderen Kreis- oder Linienelementes.
Abfrage: Sowie welcher Kreis? (Abfrage bleibt aktiv, bis eine beliebige Linie gewählt oder mit Esc abgebrochen wurde.

Stiftauswahl. Falls Stifte als Favoriten aktiviert wurden, werden diese an erster/oberster Stelle angezeigt.

1.Lage des Mittelpunktes definieren

§Geben Sie unter X-Wert und Y-Wert die Koordinaten des Mittelpunktes an oder klicken Sie die gewünschte Position an.
s. Kreis oder Teilkreis zeichnen

2.Längen der Halbachsen definieren

§Geben Sie unter Länge a (X-Achse) und Länge b (Y-Achse) die Längen der Halbachsen an.

3.Anfangs- und Endwinkel definieren

Führen Sie einen der folgenden Schritte aus:

a)Geben Sie unter Anfangswinkel den Winkel an, auf dem der Anfangspunkt des Grundkreises liegt. Bezug ist die X‑Achse.

Geben Sie unter Endwinkel einen Winkel bezogen auf die X‑Achse ein.

Länge a (X-Achse) = 1.5
Länge b (Y-Achse) = 1.5

Anfangswinkel = 45°
Endwinkel = 315°
Drehwinkel = 0°

Länge a (X-Achse) = 1.5
Länge b (Y-Achse) = 2.3

Anfangswinkel = 45°
Endwinkel = 315°
Drehwinkel = 0°

b)Wählen Sie mit einem Linksklick unter der Abfrage einen vordefinierten Voll- oder Teilkreis.

4.Drehwinkel definieren

§Geben Sie unter Drehwinkel den Winkel ein, um den die Hauptachsen gedreht werden sollen. Die Achsen stehen immer senkrecht aufeinander!
