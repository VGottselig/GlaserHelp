Textgröße in Millimeter ändern

Ändert alle markierten Texte auf eine bestimmte Größe.

Tipps:

 

·Wenn Sie die Schriftgröße eines Textfelds ändern möchten, benutzen Sie die Funktion [Konst 1 | Texte → ändern].

·Wenn Sie die Schaltfläche Belie. Größe (über der Auswahl der Schriftgröße) anklicken, können Sie die Textgröße frei wählen. Die eingegebene Textgröße wird im Auswahlbereich auf der Ziffer Nr. 8 aufgeführt und gespeichert.

So wird's gemacht:

§Führen Sie einen der folgenden Schritte aus:

a)Wählen Sie eine Schriftgröße im Funktionsbereich.

b)Übernehmen Sie die Schriftgröße eines Elementes. Klicken Sie hierfür auf und klicken Sie anschließend das Element an. [So wie welches Element?].

§Ziehen Sie einen Rahmen durch das Anklicken von zwei Punkten um die zu ändernden Zeichenelemente oder klicken Sie einzelne Elemente an. [1.Punkt/Element]; [2.Punkt].

-Die Zeichenelemente werden markiert.

-Um Elemente wieder abzuwählen, klicken Sie im unteren Funktionsbereich auf Entfer. Ziehen Sie einen Rahmen um die entsprechenden Elemente oder klicken Sie einzelne Elemente an. Bei gedrückter Umschalttaste (Shift) wird automatisch der Entfer-Modus aktiviert  am Cursor wird ein Minuszeichen eingeblendet.

§Bestätigen Sie die Auswahl mit Rechtsklick.

Die markierten Zeichenelemente werden geändert.
