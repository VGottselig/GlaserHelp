Mit festen Werten kopieren

Die markierten Elemente werden über einen Bezugspunkt neu platziert. Dreh- und Spiegelachse können angegeben werden.

1.Elemente markieren

§Ziehen Sie einen Rahmen durch das Anklicken von zwei Punkten um die zu kopierenden Zeichenelemente oder klicken Sie einzelne Elemente an. [1.Punkt/Element]; [2.Punkt].

-Die Zeichenelemente werden markiert.

-Um Elemente wieder abzuwählen, klicken Sie im unteren Funktionsbereich auf Entfer. Ziehen Sie einen Rahmen um die entsprechenden Elemente oder klicken Sie einzelne Elemente an. Bei gedrückter Umschalttaste (Shift) wird automatisch der Entfer-Modus aktiviert  am Cursor wird ein Minuszeichen eingeblendet.

§Bestätigen Sie die Auswahl mit Rechtsklick.

2.Referenzpunkt angeben

Der Referenzpunkt muss nicht im markierten Bereich liegen und sollte so gewählt werden, dass die Bestimmung der neuen Koordinaten möglichst einfach ist.

Ist der Bereich schräg angeordnet oder liegt mindestens eine Zielkoordinate auf einer schrägen Linie, ist es ratsam, das Arbeitskoordinatensystem zu drehen. Alternativ können Sie auch die Hilfsfunktion S [Schnitt] verwenden. Tippen Sie in das Eingabefeld ein s ein und klicken Sie anschließend die Schräge oder jeweils für X und Y die Schrägen an. Achten Sie darauf, bei beiden Parametern dieselbe Hilfsfunktion einzugeben!

§Bestimmen Sie den Referenzpunkt der zu verschiebenden Teilzeichnung durch Anklicken. [X-Refpu.Obj]; [Y-Refpu.Obj].

Wenn Sie zwei verschiedene Punkte anklicken, wird der X-Wert des ersten Punktes und der Y-Wert des zweiten Punktes genommen.

3.Drehwinkel und Spiegelung angeben

Führen Sie einen der folgenden Schritte aus:

a)Geben Sie den Drehwinkel (Winkel zur Horizontalen) an oder bestätigen Sie den Vorschlagswert mit Rechtsklick. Mit negativem Wert wird im Uhrzeigersinn gedreht. [Dreh.W].

b)Mit den Ausrichtungsoptionen P... oder O... können Sie die Teilzeichnung parallel oder orthogonal zu einem Bezugselement ausrichten. Klicken Sie die entsprechende Ausrichtungsoption an und klicken Sie anschließend das Bezugselement an, dessen Neigung übernommen werden soll.

c)Wenn Sie die Ausrichtungsoptionen PP oder OP verwenden, können Sie ein Element der Teilzeichnung zu einem Bezugselement ausrichten und somit die komplette Teilzeichnung drehen. [Drehen]. Klicken Sie die entsprechende Ausrichtungsoption an und anschließend ein markiertes Element. Klicken Sie im nächsten Schritt das Bezugselement an, zu dem das zuvor angeklickte (markierte) Element ausgerichtet werden soll. Die Teilzeichnung wird gedreht. Sie können die Teilzeichnung weiterdrehen. Klicken Sie auf die Schaltfläche 180°, um die Teilzeichnung in die gewünschte Position zu drehen. Wenn keine weitere Drehung erforderlich ist, bestätigen Sie mit Rechtsklick das Weiterdrehen um 0°.

§Geben Sie den Winkel an, um den gespiegelt werden soll, und bestätigen Sie mit der Eingabetaste. [Spie.W].

Wenn Sie "0" anklicken, wird um die X-Achse gespiegelt. Die Eingabe "90" spiegelt um die Y-Achse. Wenn Sie nicht spiegeln möchten, bestätigen Sie die Eingabe "-" mit Rechtsklick oder klicken Sie unterhalb der Abfrage auf Ohne.

4.Elemente platzieren

Es wird die Lage (neuer Referenzpunkt) des unter X- und Y-Refpu.Obj angegebenen Referenzpunktes für die markierten Elemente angegeben. Wenn Sie die Funktion [Retan] aktiviert haben, können Sie nach dem Anklicken der Bezugslinie über die Tastatur eine Differenz angeben.

§Klicken Sie die Koordinaten für die endgültige Lage an. [X-Refpu.Neu]; [Y-Refpu.Neu].

Die Teilzeichnung wird kopiert.

5.Noch einmal kopieren?

Sie können die kopierten Elemente nochmals an eine andere Stelle kopieren.

Führen Sie einen der folgenden Schritte aus:

a)Wählen Sie 1=ja, wenn die Markierung der Elemente beibehalten werden soll. Der Referenzpunkt wird ebenfalls übernommen.

Die Aktion wird bei [Dreh.W] wieder aufgenommen. Mit Rechtsklick können Sie gültige Werte übernehmen.

b)Wählen Sie 0=nein, um den Kopiervorgang abzuschließen und ggf. neue Elemente zu markieren.

Beispiel: Mit festen Werten kopieren

Sie können die Beispielzeichnung mit [Konst1 | Linien] oder [Konst2 | Wandze] nachzeichnen.

Eingabeeinheit: m (Meter). Bestätigen Sie alle Eingaben mit der Eingabetaste.

 

Beispiel 1:

In der Beispielzeichnung soll das Fenster kopiert werden. Wichtig ist die Hilfslinie im Bereich der Leibung. Die Hilfslinie ist mit dem gleichen Strichtyp und derselben Stiftnummer wie die Innenlinie der Vormauer gezeichnet.

1.Schalten Sie die Überlagerungskontrolle [ÜlaAn] ein.

2.Ziehen Sie einen Rahmen um das Fenster auf und bestätigen Sie mit Rechtsklick. [1. Punkt/Element]; [2. Punkt].

3.Klicken Sie den Referenzpunkt am linken Eckpunkt des Fensters an. [X-Refpu.Obj]; [Y-Refpu.Obj].

4.Überspringen Sie das Drehen und Spiegeln jeweils mit Rechtsklick, da eine Ausrichtung nicht erforderlich ist.

5.Schalten Sie den Differenzmodus [Retan] ein. [X-Refpu.Neu].

6.Für die neue Lage des Referenzpunktes klicken Sie die rechte Ecke des Fensters an.

7.Geben Sie nach der Linienkennung als horizontalen Abstand des Referenzpunktes den Wert .49 ein. Bestätigen Sie die Eingabe.

8.Übernehmen Sie die leere Eingabe der Y-Koordinate mit Rechtsklick. [Y-Refpu.Neu].

Das Fenster wird kopiert.

Beispiel 2:

In der Beispielzeichnung soll der Pfeiler kopiert werden.

1.Zeichnen Sie eine Hilfslinie vom Mittelpunkt des Teilkreises zu einem Hilfspunkt, der auf der Teilkreismitte liegt.

2.Ziehen Sie einen Rahmen um den Pfeiler auf und bestätigen Sie mit Rechtsklick. [1. Punkt/Element]; [2. Punkt].

3.Klicken Sie den Referenzpunkt (Mittelpunkt) zweimal an. [X-Refpu.Obj]; [Y-Refpu.Obj].

4.Überspringen Sie das Drehen mit Rechtsklick.

5.Um die Hilfslinie zur Spiegelachse zu machen, klicken Sie unter der Abfrage auf die Hilfsfunktion P (parallel) und klicken Sie die Hilfslinie an.

6.Bestätigen Sie den neuen Referenzpunkt an der Stelle der Zeichenmarke (Eingaben sind leer) jeweils mit Rechtsklick. [X-Refpu.Neu]; [Y-Refpu.Neu]. Der Pfeiler wird kopiert.
