E-Mail an Support senden

ISBCAD Menüleiste → Support

ISBCAD kann eine E-Mail an den ISBCAD Kundensupport erstellen, an die ihre aktuelle Zeichnung, das verwendete Profil sowie eine Protokolldatei angehängt werden kann.

Um eine effiziente Bearbeitung Ihres Problems zu gewährleisten, geben Sie in der E-Mail eine kurze Problembeschreibung ein und seien Sie spezifisch. Fügen Sie ggf. weitere Dateien und Screenshots​ bei, die hilfreich sein könnten. Je mehr Informationen Sie uns mitteilen, desto schneller können wir Ihre E-Mail beantworten, das Problem beheben oder Hilfe anbieten.

Für weitere Hilfe rufen Sie bitte den ISBCAD Kundensupport an. Sie erreichen unseren Kundensupport von Montag bis Freitag von 09:00 - 12:00 Uhr sowie von 14:00 - 17:00 Uhr unter +49 511 592931-0.

So wird's gemacht:

§Klicken Sie in der ISBCAD Menüleiste unter [Support] den Eintrag [E-Mail an Support senden ...] an.

Anschließend können Sie auswählen, welche Dateien der E-Mail angehängt werden sollen. Setzen Sie dazu einen Haken in die Checkbox.

·aktuelle Zeichnung: Die aktuelle Zeichnung wird gepackt und der E-Mail im ZIP-Format angehängt.

·aktuelles Profil: Das aktuell verwendete Profil der Zeichnung wird angehängt. Wenn das Basisprofil verwendet wird, ist diese Option deaktiviert.

·Log-Datei: Der E-Mail wird die Protokolldatei "isbcad.log" im ZIP-Format angehängt. Diese Datei kann zwecks Fehleranalyse an den ISBCAD-Kundensupport geschickt werden und wird ausschließlich zur Bearbeitung einer Supportanfrage genutzt.

·Systeminformationen: Der E-Mail wird eine Textdatei mit Systeminformationen angehängt. Darin enthalten sind Details zum PC, zum Betriebssystem und zur Hardware einschließlich Treibern.

§Klicken Sie auf OK.

Die E-Mail wird als neue Nachricht in Ihrem Standard-E-Mail-Client (z. B. Outlook, Thunderbird, Windows 11 Mail-App etc.) geöffnet.

Die Empfängeradresse (support@glasercad.de) und der Betreff mit Ihrer ISBCAD Version, der Revisionsnummer und Ihrer Kundennummer sind bereits eingetragen. Die gewählten Dateien wurden dem Anhang hinzugefügt.

Beispiel: Microsoft Outlook

§Bevor Sie die E-Mail versenden, beschreiben Sie bitte das Problem. Fügen Sie ggf. Screenshots bei, die das Problem verdeutlichen.
