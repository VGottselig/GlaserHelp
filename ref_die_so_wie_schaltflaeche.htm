Die so wie-Schaltfläche

Mit der so-wie-Funktion können Sie die Attribute des angeklickten Elements übernehmen. Beachten Sie, dass Attribute nur von Elementen gleichen Typs übernommen werden können.

So wird's gemacht:

§Klicken Sie im Funktionsbereich auf .

§Klicken Sie ein Element an, dessen Attribute Sie übernehmen wollen. [So wie welches...].

Die Attribute des gewählten Elements werden in die entsprechenden Auswahloptionen (z. B. Stiftauswahl) übertragen. Sie können die aktuelle Funktion (z. B. Linien zeichnen) jetzt mit den übernommenen Attributen fortsetzen.

Folgende Elementattribute werden übernommen:

Elementtyp

Übertragene Attribute

Linien, Kreise

Stiftnummer und Strichtyp

Texte

Textauszeichnungen, Stiftnummer, Schriftart, Textgröße und Breitenfaktor

Maßketten

Stil, mit/ohne Gesamtmaß, Einheiten

Schraffur

Schraffurtyp, Stiftnummer, Faktor und Winkel

Filling (Füllung)

Stiftnummer

Schnittpfeile

Schnittpfeil-Symbol, Stiftnummer, Schriftart, Textgröße und Breitenfaktor

Wolke

Stiftnummer und Texteinstellungen (Schriftart, Textgröße, Breitenfaktor)

Hüllfläche

Bauteiltyp, Himmelsrichtung, Neigung, Farbe, Umrandungsbreite, Textdarstellung

Stat. Positionskreise

Positionskreis-Stil (Text, Linien und Pfeile)

Flächenmatten (VerEin)

Alle Parameter der angeklickten Matte

Positionierungen

Stilnummer und Symbole

Verlegungen

Durchmesser, Abstand oder Anzahl
