Zylindrische Bauteile bewehren

Alle Inhalte aufklappen/schließen

Kreisförmige Bewehrung zylindrischer Bauteile mit Verteilung über die Höhe.

Die Bearbeitung erfolgt schrittweise anhand der freigeschalteten Optionen. Nach Angabe der Geometrie haben Sie die Möglichkeit, einzelne Parameter zu definieren.

Um die Bewehrung zu übernehmen, muss sie abschließend bestätigt werden. Wenn Sie die Bearbeitung unterbrechen, z. B. durch Anklicken eines anderen Programmteils, wird die nicht abgeschlossene Bewehrung ggf. gelöscht. Bestätigen Sie die Abfrage mit Ja, um die Bearbeitung wieder aufzunehmen.

1.Geometrie angeben

§Geben Sie unter Randabstand den Wert für die Betonüberdeckung ein bzw. bestätigen Sie die Angabe mit der Eingabetaste.

§Klicken Sie das zu bewehrende Bauteil (Kreis) an. [Kreis ankl.].

§Geben Sie nacheinander den Startwinkel und den Endwinkel der Bewehrung ein bzw. bestätigen Sie die vorhandenen Werte mit der Eingabetaste.
Für eine Vollkreisbewehrung: Start-Winkel: 0; End-Winkel: 360

§Geben Sie die Längenmaße an bzw. bestätigen Sie die vorhandenen Werte mit der Eingabetaste.

Stab-Einzellänge

Länge der Einzelstäbe.

Übergreifung

Übergreifung der Einzelstäbe.

Rest

Erforderliche Restlänge. Vorgabewert wird automatisch aus Stab-Einzellänge und Übergreifung ermittelt.

–Die Bewehrung wird in der Zeichnung angezeigt.

–In der Abfrage wird "99" angeschrieben. Damit wird verhindert, dass Sie durch einen versehentlichen Rechtsklick die Eingaben beenden. [1=Verlegung OK]. Diese Abfrage wird im Folgenden nach jeder optionalen Parameteranpassung eingeblendet. Wenn Sie die Bewehrung beenden möchten, können Sie durch Anklicken von Ende/OK oder der Eingabe von 1 die Bewehrung abschließend bestätigen (s. Schritt 3)

–Um Parameter anzupassen: Wählen Sie die nächste Funktion im Funktionsbereich. Wir empfehlen, die Funktionen nacheinander abzuarbeiten. Wenn Sie sich sicher sind, dass in den einzelnen Funktionen keine Änderungen vorgenommen werden müssen, können Sie die entsprechenden Funktionen überspringen.

2.Parameter anpassen (optional)

Bauteilhöhe eingeben

§Klicken Sie auf .

§Geben Sie unter Zylinderhöhe die Höhe des Bauteils ein bzw. bestätigen Sie den vorhandenen Wert mit der Eingabetaste.

Sie können jetzt weitere Parameter anpassen oder mit Ende/OK die Bewehrung abschließend bestätigen.
Stabdurchmesser und Abstand der Stäbe ändern

§Klicken Sie auf .  

§Geben Sie unter ø den gewünschten Durchmesser und unter a den Abstand der Stäbe ein bzw. bestätigen Sie vorhandenen Werte mit der Eingabetaste.

Sie können jetzt weitere Parameter anpassen oder mit Ende/OK die Bewehrung abschließend bestätigen.
Auszüge verschieben

Beim Anlegen einer Bewehrung werden automatisch Auszüge erstellt. Die Auszugsteile (Form und Auszugstext) können nacheinander verschoben werden.

Das Programm nimmt automatisch der Reihe nach jeden Auszug und jeden Auszugstext an den Cursor, so dass Sie lediglich die endgültige Lage bestimmen müssen.

§Klicken Sie auf .

§Klicken Sie an die Stelle, an der die aktuelle Form bzw. der aktuelle Auszugstext erscheinen soll.

§Wiederholen Sie den Vorgang, bis alle Auszüge an der gewünschten Stelle platziert sind.

–Mit Esc können Sie die Auszüge wieder an die ursprüngliche, vom Programm vorgesehene Stelle zurücksetzen.

–Mit Rechtsklick übernehmen Sie die vorgeschlagene Lage.

–Sie können jetzt noch weitere Parameter anpassen oder mit Ende/OK die Bewehrung abschließend bestätigen.

3.Bearbeitung abschließen

§Klicken Sie auf oder geben Sie unter 1=Verlegung OK den Wert 1 ein und bestätigen Sie mit der Eingabetaste.
