Dübelleiste an einer Kreisstütze verlegen

Die Dübelleisten werden an einem Kreis verlegt. Freie Ränder können entweder über das Anklicken von Linien definiert werden oder durch die Eingabe eines Winkels.

1.Kreis wählen

§Klicken Sie den Kreis an. [Welcher Kreis?].

Ein Teilkreis kann nicht angeklickt werden.

§Wenn Sie Randbedingungen angegeben haben, führen Sie für den ersten oder auch zweiten Rand einen der folgenden Schritte aus. [1./2. freier Rand/Winkel].

a)Klicken Sie die Begrenzungslinie(n) an, zu der / zu denen hin keine Dübelleisten verlegt werden sollen. Die Begrenzungslinien werden markiert und mit einem gekennzeichnet. Der Kreis wird mit der voreingestellten Anzahl an Dübelleisten dargestellt.

b)Geben Sie den Winkel manuell ein oder klicken Sie eine Winkelvorgabe an.

Definition von Rändern durch Winkeleingaben

Sie können die Dübelleisten in einem bestimmten Winkelbereich verlegen, indem Sie Randbedingungen angeben. Die Ränder entsprechen einer Tangente, die im angegebenen Winkel an den Kreis gelegt wird. An diesen Rändern werden keine Dübelleisten verlegt (= freier Rand).

(MP) = Mittelpunkt

Bei aktiven Randbedingungen werden die Dübelleisten gleichmäßig zwischen der ersten und letzten Dübelleiste verlegt. Der Winkel der Ränder muss mindestens 45° betragen, ansonsten kann die zweite Randbedingung nicht angegeben werden.

Beispiel:

a) 1. freier Rand/Winkel = 315° (manuelle Eingabe).

b) 2. freier Rand/Winkel = Winkelvorgabe 90° angeklickt.

2.Parameter anpassen

§Geben Sie im Dialog die Anzahl der Dübelleisten an. Die Änderung wird auf der Zeichnung aktualisiert.

Sie können die Dübelleistendefinition ändern. Klicken Sie auf Ändern, um den Dialog Eigenschaften Dübelleisten zu öffnen. Geben Sie im Dialog die neuen Parameter an und bestätigen Sie mit Ok. Der gewählte Dübelleistentyp wird im Bereich Angewandte Dübelleiste angezeigt.

§Bestätigen Sie die Verlegung mit OK.

Der Dialog Auswahl Bauteiltyp und Randbedingung wird geöffnet. Sie können weitere Dübelleisten verlegen oder den Dialog mit Abbrechen schließen.

Zugehörige Aufgaben:

·Dübelleiste beschriften

·Dübelleiste im Schnitt verlegen

·Dübelleiste löschen

·Dübelleistentyp definieren
