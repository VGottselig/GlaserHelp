Hilfsfunktionen

Verfügbar im Zeichenmodus ohne Richtungslogik [oRiL] und im Bewehrungsprogramm.

Bei der Eingabe der X- bzw. Y-Werte können Hilfsfunktionen zur Vereinfachung der Konstruktion verwendet werden. Hierdurch können auch komplexe Konstruktionen ohne die Eingabe von Hilfspunkten und Hilfslinien realisiert werden. Einzelne Werte oder Berechnungen in den Eingabefeldern werden bereits während der Eingabe durch Konstruktionslinien auf der Zeichenfläche dargestellt. Die Hilfsfunktionen lassen sich auch miteinander kombinieren. Sie können somit, je nach Bedarf, für die X- und Y- Eingabe die entsprechende Hilfsfunktion wählen.

Nutzen Sie die Hilfsfunktionen auch für das Platzieren der Zeichenmarke an einem bestimmten Punkt.

Hilfsfunktion aufrufen

Um eine Hilfsfunktion aufzurufen, geben Sie das entsprechende Kürzel in die Eingabe ein, z. B.: w (für Winkel) oder p (für das Zeichnen einer Parallelen). Für die Tastenkürzel können Sie einen Klein- oder Großbuchstaben eingeben. Sie können auch den Cursor direkt unter das Eingabefeld der Abfrage bewegen, um das Kontextmenü mit den Hilfsfunktionen zu öffnen. Wenn Sie auf eine Funktion klicken, wird die Auswahl in das Eingabefeld übernommen. Die Hilfsfunktionen x-Koordinate, y-Koordinate und Lotfußpunkt können nur über das Y-Eingabefeld aufgerufen werden.

Tipp: Die Hilfsfunktionen können Sie auch mit dem Differenzmodus [Retan] kombinieren. Dadurch können Sie nach der Wahl einer Hilfsfunktion und einem Klick auf eine Linie oder einen Punkt Differenzangaben zum angeklickten Zeichenelement eingeben.

Im Bewehrungsprogramm ist der Differenzmodus standardmäßig eingestellt.

Hilfsfunktionen

·Winkel (w) - Linien von der Zeichenmarke aus mit einer bestimmten Neigung zeichnen.

·Schnitt (s) - Schnittpunkt zweier Linien bestimmen. An schrägen Linien zeichnen und Zeichenmarke setzen.

·Orthogonale (o) - Eine Senkrechte von der Zeichenmarke aus zu einer Bezugslinie zeichnen.

·Parallele (p) - Eine Parallele zu einer Bezugslinie zeichnen.

·Tangente (t) - Eine Tangente an einen Kreis, Teilkreis oder Bogen zeichnen.

·x-Koordinate (x) - Den X-Wert eines angeklickten Punktes übernehmen.

·y-Koordinate (y) - Den Y-Wert eines angeklickten Punktes übernehmen.

·Lotfußpunkt (l) - Einen Lotfußpunkt zu einer Bezugslinie zeichnen.
