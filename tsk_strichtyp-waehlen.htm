Strichtyp auswählen

Die Strichtyp-Auswahl im Funktionsbereich steht immer dann zur Verfügung, wenn für die aktuell gewählte Funktion verschiedene Strichtypen ausgewählt werden können.

Strichtyp-Auswahl

Tipps:

 

·Sie können jederzeit, auch während einer laufenden Eingabe, einen anderen Strichtyp auswählen.

·Strichtypen können im Dialog Systemdaten konfiguriert werden; [Option | SysDat | Strichtypen].

So wird's gemacht:

§Klicken Sie auf die Ausklappliste und wählen Sie den gewünschten Strichtyp.

Verwenden Sie zum Scrollen in der Liste nur das Mausrad. (Mit den Pfeiltasten (Tastatur) wird die Hilfspunkte- oder Messen-Funktion aufgerufen).

§Wenn die Nummer des Strichtyps bekannt ist, können Sie diese auch direkt anklicken. Verwenden Sie zum Durchblättern der Strichtypnummern die Pfeile neben der Ausklappliste.
