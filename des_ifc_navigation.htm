Navigation im Modellbereich

Im Modellbereich des IFC Section Tools sehen Sie das Modell von Ihrem Standpunkt aus. Dieser Standpunkt wird durch eine Kamera repräsentiert, die im 3D-Raum bewegt wird. Das digitale Modell lässt sich dabei wie ein reales Modell aus verschiedenen Perspektiven und mit unterschiedlichen Zoomstufen betrachten. Sie können das Modell z. B. um 360° drehen oder nach oben und unten kippen, um eine bestimmte Ansicht einzustellen.

Kamera ausrichten

Im IFC Section Tool stehen Ihnen verschiedene Methoden zur Verfügung, mit denen Sie die Ansicht des Modells anpassen können.

In der Symbolleiste können Sie eine Funktion aktivieren, indem Sie das Symbol anklicken. Zum Deaktivieren klicken Sie das Symbol erneut an.

Drehen

Maussteuerung

Klicken und halten Sie die rechte Maustaste. Bewegen Sie anschließend die Maus, um die Ansicht neu auszurichten.

Wenn Sie die Maus ziehen, dreht sich das Modell um den Drehpunkt. Die Ansicht bleibt jedoch fixiert. Lassen Sie die Maustaste los, um an einem neuen Drehpunkt anzusetzen.

Zoomen

Ansichtswürfel

Der Ansichtswürfel ist ein Navigationswerkzeug, das angeklickt und gezogen werden kann. Während Sie im Modellbereich navigieren, gibt Ihnen der Würfel bei Ansichtsänderungen visuelles Feedback zum aktuellen Ansichtspunkt des Modells.

Standardansichten

Wenn Sie den Cursor auf dem Ansichtswürfel platzieren, wird es aktiviert. Klicken Sie eine Seitenfläche oder eine Kante an, um das Modell neu auszurichten.

Seitenfläche

Kante

Sie können auch eine Tastenkombination nutzen, um eine bestimmte orthogonale Ansicht einzustellen. Alternativ lässt sich die Ansicht auch über die Menüleiste unter [Ansicht → Standard-Ansichten] einstellen.

Ausrichten des Modells

Wenn Sie die linke Maustaste auf dem Ansichtswürfel gedrückt halten und die Maus bewegen, wird das Model gedreht und geneigt.

Um das Modell in der aktuellen Ansicht um einen festen Punkt zu drehen, klicken und ziehen Sie den Ring.
